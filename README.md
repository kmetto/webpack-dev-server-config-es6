# Webpack config example written in es6 syntax

## Launch webpack-dev-server

```bash
git clone https://github.com/kmetto/webpack-dev-server-config-es6.git
cd webpack-dev-server-config-es6
npm i
npm run serve
```

## Or build dist

```bash
git clone https://github.com/kmetto/webpack-dev-server-config-es6.git
cd webpack-dev-server-config-es6
npm i
npm run build
```
